aModels Chromium Extension

The extension shipped in services/browser/extension/ provides gateway shortcuts, LocalAI chat, and basic system checks. Load it unpacked into any Chromium-based browser while iterating on the UI.

Load Unpacked

  • Open chrome://extensions and enable Developer Mode.
  • Choose “Load unpacked” and point at services/browser/extension/.
  • Pin the toolbar icon to keep gateway utilities handy.

Gateway Integrations

Configure the base URL on the options page (chrome-extension://<ID>/options.html). The popup can then run health checks, AgentFlow demos, telemetry queries, and LocalAI chat through the Layer4 gateway.

  • /healthz and /telemetry/recent for status snapshots.
  • /agentflow/run, /data/sql, and Redis helpers for demo flows.
  • /localai/chat to proxy messages to LocalAI-compatible models.
